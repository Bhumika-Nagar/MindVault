# 🧠 MindVault

MindVault is a **second brain web application** built using the **MERN stack with TypeScript**, designed to help users save, organize, and revisit valuable content from across the internet. It enables users to curate their own knowledge hub and generate shareable links for easy access and collaboration.

## 🚀 Features

* 🔗 Save useful resources (articles, videos, audio)
* 🧾 Structured content storage with type-based categorization
* 🌐 Generate public shareable links for saved content
* 🧠 Build a personal knowledge vault for long-term learning
* 🗑️ Delete and manage saved content easily
* 🔐 Secure authentication using JWT + bcrypt

## 🛠️ Tech Stack

### Frontend

* React (with TypeScript)
* Vite
* Tailwind CSS (or similar styling)

### Backend

* Node.js
* Express.js
* TypeScript

### Database

* MongoDB

### Authentication

* JWT (JSON Web Tokens)
* bcrypt (password hashing)

## 🧠 How It Works

1. User signs up / logs in securely using JWT-based authentication
2. Selects a content type (Article / Video / Audio)
3. Adds a title and resource link
4. Content is stored in MongoDB
5. User can:

   * View saved content
   * Delete entries
   * Generate a public shareable brain link
6. Shared link allows others to view curated content
